# Zainu VIM
Provides a bunch of unrelated helpers

~~~
<leader>+x+i  => Toggles Zoom. Opens current window in new tab, when you hit it again it closes this new tab
<leader>+x+x  => Opens a scratch file for the current project
            If you use multiple computers you can sync this scratch folder
            using dropbox and have a nice and simple way to sync your scratch
            data.
<leader>+x+t  => Opens a quick todo file
~~~
